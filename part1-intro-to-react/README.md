# Part 1 — Introduction to React

Introduction to React and the JavaScript fundamentals underpinning it:
components, props, state, event handling, and debugging React applications.

## Key Concepts
- React components and JSX syntax
- Core JavaScript (ES6+): arrow functions, destructuring, spread, classes, promises
- Props and component composition
- Component state with useState
- Event handlers and re-rendering
- Complex state (objects and arrays)
- Conditional rendering and debugging

## Projects & Exercises
- **Course Info:** Renders a structured breakdown of course parts and exercise
  counts using component composition and props
- **Unicafe:** A real-time feedback app tracking ratings, computing averages
  and displaying statistics
- **Anecdotes:** Displays random programming anecdotes and tracks user votes

## Technologies
React, JavaScript (ES6+), Vite, Node.js v22
