# Part 3 — Node.js, Express & MongoDB

Builds the backend for the Part 2 Phonebook app using Node.js and Express,
connects it to MongoDB, deploys the full-stack application to Render,
and adds validation and linting.

## Key Concepts
- Node.js and npm fundamentals
- Express REST API design and middleware
- File watching with `node --watch`
- Connecting a React frontend to a Node.js backend (proxy configuration)
- MongoDB Atlas setup and Mongoose schemas and models
- Data validation with Mongoose
- Full-stack deployment to Render
- ESLint configuration and code quality

## Projects & Exercises
- **Phonebook — Full Stack:** REST API with full CRUD for contacts,
  MongoDB persistence, Mongoose validation, ESLint,
  and a production full-stack deployment — exercises 3.1–3.22
