# Part 4 — Testing Express Servers & User Authentication

Covers backend testing, refactoring Node.js apps for testability, and building
a complete JWT-based user authentication system using Express v5.

## Key Concepts
- Node.js project structure best practices
- Unit and integration testing with Node:test
- Async/await patterns and Express v5 native async error handling
- User administration and resource ownership
- JWT-based token authentication
- Password hashing with bcrypt

## Projects & Exercises
- **Blog List API:** REST API for managing blog posts with a full test suite
  (unit and integration), user registration and login endpoints,
  and token-protected create and delete routes — exercises 4.1–4.22

## Technologies
Node.js v22, Express v5, MongoDB, Mongoose, Node:test, JSON Web Tokens, bcrypt
