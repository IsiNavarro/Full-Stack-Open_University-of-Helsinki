# Part 0 — Fundamentals of Web Apps

Covers how web applications work under the hood: HTTP request/response cycles,
browser rendering, traditional vs single-page applications, and JSON data exchange.

## Key Concepts
- HTTP GET and POST requests
- Browser developer tools and the network tab
- Traditional server-rendered pages vs single-page applications
- JSON as a data interchange format
- Sequence diagrams for mapping application flows
