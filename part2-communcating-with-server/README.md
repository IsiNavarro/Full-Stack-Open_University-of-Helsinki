# Part 2 — Communicating with a Server

Covers rendering dynamic server data, form handling, and full CRUD operations
via a REST API. Introduces axios for HTTP requests and json-server for local
backend simulation.

## Key Concepts
- Rendering collections with map and key props
- Controlled forms and input handling
- Fetching data with axios and useEffect
- REST API conventions (GET, POST, PUT, DELETE)
- Error handling and in-app user notifications
- Inline and external styling in React

## Projects & Exercises
- **Course Info (extended):** Adds cumulative exercise count rendering to the
  Part 1 project
- **Countries:** Fetches and filters country data from a public REST API,
  with live weather data via the OpenWeatherMap API
- **Phonebook:** Full CRUD contact manager with server persistence, live
  filtering, update and delete operations, and user notifications

## Technologies
React, JavaScript (ES6+), Axios, REST APIs, json-server
